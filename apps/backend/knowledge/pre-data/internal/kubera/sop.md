# KUBERA Department - Standard Operating Procedures

## Daily Operations

### Morning Review (08:30 UTC)
1. Review resource utilization
2. Check pending procurement requests
3. Monitor operational metrics
4. Review vendor performance
5. Track cost trends

## Resource Allocation SOP
1. Receive resource request
2. Check availability
3. Validate justification
4. Get approval if needed
5. Allocate resource
6. Track utilization
7. Reclaim if underutilized
